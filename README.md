# Codespace Template for UET Learning

This repository contains a minimal devcontainer that reproduces a CS50-like environment suitable for student exercises.

Files:
- `.devcontainer/devcontainer.json` — Codespaces / VS Code devcontainer configuration
- `.devcontainer/Dockerfile` — Dockerfile installing system packages
- `requirements.txt` — Python packages installed in post-create step
- `setup.sh` — optional project setup script

Usage
- Mark this repository as a template or keep it in your org. Users will be forked from it automatically and a Codespace created for their fork.
